## What This Site Shows

This portfolio is built as a fast, single-page GitHub Pages website with a cyber-lab visual identity. It presents my work across web development, AI, embedded systems, and ethical cybersecurity research.

Highlights include:

- A modern full-screen hero section with project visuals
- Featured projects including Privy.AI, CyberVision Glasses, NeoBMO, CodeFlux AI, AirMouse, and WatchGuard
- Experience timeline with freelance, full-stack, AI integration, and Bursa Web Pro work
- Education section for BSc Artificial Intelligence Engineering at Oxford Brookes University
- Fixed CV download button connected to `cv.pdf`
- Responsive layout for desktop and mobile
- Lightweight HTML, CSS, and JavaScript with no build step

## CV Download

The site includes a fixed CV button in the bottom-left corner.

```html
<a class="cv-download" href="cv.pdf" download>
```

For the button to work on GitHub Pages, `cv.pdf` must stay in the same folder as `index.html`.
